# OG Image для социальных сетей

## Требования к изображению

Создайте файл `og-image.jpg` в этой папке со следующими параметрами:

### Размеры
- **Ширина**: 1200px
- **Высота**: 630px
- **Формат**: JPG или PNG
- **Соотношение сторон**: 1.91:1

### Содержание
Изображение должно включать:
- Логотип YouWow
- Текст: "Персональные подарки с WOW-эффектом"
- Подзаголовок: "Видео-поздравления и персональные песни"
- Яркий, привлекательный дизайн с градиентами

### Где используется
- Open Graph (Facebook, LinkedIn, VK)
- Twitter Cards
- Мессенджеры при расшаривании ссылки
- Социальные сети

### Инструменты для создания
- Canva (шаблоны для OG images)
- Figma
- Adobe Photoshop
- Online генераторы OG изображений

### Проверка
После создания протестируйте на:
- https://www.opengraph.xyz/
- https://cards-dev.twitter.com/validator
- Facebook Sharing Debugger
