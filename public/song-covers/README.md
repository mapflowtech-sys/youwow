# Song Cover Images

This folder contains cover images for song videos.

## Preset Covers

Place 6 preset cover images in the `presets/` folder:
- `preset-1.jpg` - Новогодняя тематика
- `preset-2.jpg` - Романтическая тематика
- `preset-3.jpg` - День рождения
- `preset-4.jpg` - Праздничная
- `preset-5.jpg` - Музыкальная тематика
- `preset-6.jpg` - Абстрактная

**Recommended specifications:**
- Format: JPEG or PNG
- Resolution: 1280x720 (720p) or 1920x1080 (1080p)
- Aspect ratio: Square (1:1) preferred
- Max file size: 5 MB per image

## Custom Covers

User-uploaded covers are temporarily stored and processed on the server.
